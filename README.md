# DNA-Anyalist-Toolkit-
Analyzes DNA sequences by calculating GC-content, transcribing DNA to RNA, and locating the ATG start codon.
How it works: 
When you run the program, it prompts the user to enter a sequence and outputs the biological analysis. 
Example: 
Enter a DNA sequence: atgctagctagctagctaggctaa.
The number of Guanine in the DNA sequence is 6.
The number of Cytosine in the DNA sequence is 5.
The total length of the DNA sequence is 24.
The GC percentage of the DNA sequence is 45.83%.
The RNA sequence is AUGCUAGCUAGCUAGCUAGGCUAA.
The position of the start codon is 0.
